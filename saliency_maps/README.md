# Saliency Maps

This folder contains the saliency outputs used for qualitative and quantitative
analysis.

## Folders

- `experiment2/`: Grad-CAM maps for the scene-only Places365 model.
- `experiment3/`: Grad-CAM maps and face/body/background saliency mass for the
  face + scene fusion model.

All saliency examples are sampled from the CAER-S in-domain test split.
